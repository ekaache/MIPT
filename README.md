# Цель
Разработать систему для автоматического сбора данных о книгах с сайта Books to Scrape.
Реализовать функции для парсинга всех страниц сайта, извлечения информации о книгах, 
автоматического ежедневного запуска задачи и сохранения результата.

# Инструкции по запуску
1. Установите зависимости:
pip install -r requirements.txt
2. Запустите парсер:
python scraper.py

# Дополнительные команды:
1. Запуск тестов
pytest tests/
2. Запуск с автоматическим расписанием (каждый день в 19:00)
python scheduler.py

# Список используемых библиотек:
requests
schedule
beautifulsoup4
pytest
